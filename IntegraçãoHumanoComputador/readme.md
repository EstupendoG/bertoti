# ATIVIDADE 1

## CLI

A <i>Command Line Interface</i> era a interface do usuário padrão dos primeiros computadores modernos, baseado somente em linhas de comando. O fato do usuário ter que escrever os comandos que serão executados pela máquina resulta em um grande poder computacional, porém com pouca acessibilidade, uma vez que depende do conhecimento do usuário em relação aos comandos envolvidos para funcionar.
Era comum nos primeiros sistemas operacionais, como o famoso Unix. 

## GUI
A popularização dos computadores para o público leigo, em parte, se da por conta da simplificação da interface do usuário, que facilitava seu uso. A chamada <i>Graphical User Interface</i> surge com o computador Xerox Alto, apresentando um sistema de ícones e janelas. 
Como o nome evidencia, esse sistema de interface foi posteriormente adotado pelo Windows.

## NUI
O potencial de uma <i>Natural User Interface</i> é a facilidade de um usuário principiante e completamente leigo dominar rápida e facilmente seu funcionamento. Se baseia em movimentos naturais, como rotacionar o dispositivo para que o conteúdo em sua tela seja rotacionado também.
A simplicidade desse tipo de interface resulta em usuários de poucos anos de vida dominando seu funcionamento.
Um dos exemplos mais famosos de <i>NUI</i> é o iPhone 1.
